# User Type Analysis — getfierro.com

Objective review of getfierro.com from the perspective of four potential user types.
Conducted March 2026 by reviewing all public pages: homepage, /why-fierro, /pricing, /privacy, /terms, and signup/login flows.

## Summary Matrix

| User Type | Would Convert? | Biggest Blocker | Severity |
|-----------|---------------|-----------------|----------|
| [Homeowner](./homeowner.md) | No | Site doesn't speak to them at all | High |
| [Flipper / Investor](./flipper-investor.md) | No | No ROI story, no proof, limiting tiers | High |
| [General Contractor](./general-contractor.md) | No | No social proof, no change order support | Critical |
| [Builder (Production/Custom)](./builder.md) | No | Entirely wrong product category | Critical |

## Universal Issues (Affect All User Types)

### Trust
- Zero testimonials or social proof of any kind
- No team page, no company story, no faces
- No social media presence linked
- No product screenshots or demo video
- No phone number — email only

### Product Visibility
- Users must sign up blind — no screenshots, no video, no interactive demo
- "See How It Works" leads to a 3-step text explanation, not a walkthrough
- No mobile app evidence (App Store / Play Store links)

### Legal Concerns
- ToS allows termination "with or without cause, with or without notice"
- Privacy Policy disclaims all AI/MCP liability onto the user
- Both flagged as aggressive by any legal review

### Positioning
- Site doesn't clearly differentiate from Buildertrend, CoConstruct, Procore, or spreadsheets
- AI/MCP is positioned as a technology feature rather than a user benefit
- No competitive comparison or "why switch" content

## Recommended Priority Actions

### Immediate (This Week)
1. Add 3-5 product screenshots to homepage
2. Add a 60-second walkthrough video
3. Get 3 beta user testimonials live
4. Add an "About" or "Our Story" section

### Short Term (This Month)
5. Create user-type-specific landing pages or sections
6. Reframe AI features as benefits, hide technical details
7. Add integrations info (even if it's "coming soon")
8. Show Builder tier starting price
9. Add change order tracking to feature messaging (if it exists in product)
10. Add a phone number or live chat

### Medium Term (This Quarter)
11. Publish 3-5 blog posts showing construction industry knowledge
12. Add case studies with real numbers
13. Build an interactive demo or sandbox
14. Get SOC 2 or equivalent security certification
15. Soften ToS termination clause
