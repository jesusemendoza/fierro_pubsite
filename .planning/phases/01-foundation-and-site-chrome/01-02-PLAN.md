   - "Start Free" CTA: ghost/outline button -- `border border-molten-orange text-molten-orange hover:bg-molten-orange hover:text-white` with rounded corners and padding. Links to `https://app.getfierro.com/signup`
   - Hamburger button (visible on mobile only, `md:hidden`): 44px minimum touch target, three horizontal lines icon, aria-label="Toggle menu"

   **Inline script** (inside Nav.astro):
   ```html
   <script>
     document.addEventListener('astro:page-load', () => {
       const nav = document.getElementById('main-nav');
       const toggle = document.getElementById('menu-toggle');
       const mobileMenu = document.getElementById('mobile-menu');
       const menuClose = document.getElementById('menu-close');

       // Scroll-based background transition
       const handleScroll = () => {
         if (window.scrollY > 50) {
           nav?.classList.add('bg-gunmetal');
           nav?.classList.remove('bg-transparent');
         } else {
           nav?.classList.remove('bg-gunmetal');
           nav?.classList.add('bg-transparent');
         }
       };
       window.addEventListener('scroll', handleScroll, { passive: true });
       handleScroll();

       // Mobile menu toggle
       const openMenu = () => {
         mobileMenu?.classList.remove('hidden');
         document.body.classList.add('overflow-hidden');
       };
       const closeMenu = () => {
         mobileMenu?.classList.add('hidden');
         document.body.classList.remove('overflow-hidden');
       };
       toggle?.addEventListener('click', openMenu);
       menuClose?.addEventListener('click', closeMenu);

       // Close mobile menu when a link is clicked
       mobileMenu?.querySelectorAll('a').forEach(link => {
         link.addEventListener('click', closeMenu);
       });
     });
   </script>
   ```
   Use `astro:page-load` event (NOT top-level script) so it re-runs after ClientRouter page transitions.
