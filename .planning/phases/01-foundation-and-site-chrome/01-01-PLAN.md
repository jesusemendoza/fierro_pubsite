<task type="auto">
  <name>Task 1: Scaffold Astro 5 project with Tailwind v4 and DM Sans</name>
  <files>package.json, astro.config.mjs, tsconfig.json, src/styles/global.css</files>
  <action>
1. Initialize an Astro 5 project in the current directory (fierro_pubsite). Since `src/assets/` already exists with logo.svg and favicon.ico, do NOT use `npm create astro` (it would overwrite). Instead, manually create the project files:

   - Create `package.json` with:
     - name: "fierro-pubsite"
     - scripts: { "dev": "astro dev", "build": "astro build", "preview": "astro preview" }
     - dependencies: astro (^5.18), tailwindcss (^4.0), @tailwindcss/vite (^4.0), @fontsource-variable/dm-sans (latest)
     - devDependencies: typescript (^5.7)

   - Run `npm install` to install all dependencies.

2. Create `astro.config.mjs`:
   ```js
   import { defineConfig } from 'astro/config';
   import tailwindcss from '@tailwindcss/vite';

   export default defineConfig({
     site: 'https://getfierro.com',
     vite: {
       plugins: [tailwindcss()],
     },
   });
   ```

3. Create `tsconfig.json` with Astro's "strict" preset:
   ```json
   { "extends": "astro/tsconfigs/strict" }
   ```

4. Create `src/styles/global.css` with:
   ```css
   @import "tailwindcss";
   @import "@fontsource-variable/dm-sans";

   @theme {
     --color-gunmetal: oklch(0.235 0.006 265);
     --color-molten-orange: oklch(0.62 0.19 48);
     --color-off-white: oklch(0.97 0.005 90);
     --color-concrete-gray: oklch(0.63 0.006 265);
     --color-rebar-green: oklch(0.53 0.12 160);
     --color-overrun-red: oklch(0.55 0.22 25);

     --font-sans: 'DM Sans Variable', system-ui, sans-serif;
   }
   ```

IMPORTANT anti-patterns to avoid:
- Do NOT install @astrojs/tailwind (deprecated for Tailwind v4). Use @tailwindcss/vite directly.
- Do NOT create tailwind.config.js (not needed with Tailwind v4 CSS-first config).
- Do NOT use Google Fonts CDN. Fontsource is self-hosted WOFF2.
  </action>

  </verify>

1. Create `src/layouts/BaseLayout.astro`:

   - Render full HTML document:
     ```

       <head>
         <meta charset="utf-8" />
         <meta name="viewport" content="width=device-width, initial-scale=1" />
         <link rel="icon" href="/favicon.ico" />
         <title>{title} | Fierro</title>
         <ClientRouter />
       </head>
       <body class="bg-off-white text-gunmetal font-sans antialiased">
         <slot />
       </body>
     </html>
     ```
