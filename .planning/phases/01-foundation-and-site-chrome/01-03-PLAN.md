<task type="auto">
  <name>Task 1: Configure Cloudflare Workers Static Assets deployment</name>
  <files>wrangler.jsonc, package.json</files>
  <action>
1. Install wrangler as a dev dependency:
   ```bash
   npm install -D wrangler
   ```

2. Create `wrangler.jsonc`:
   ```jsonc
   {
     "name": "fierro-pubsite",
     "compatibility_date": "2026-03-09",
     "assets": {
       "directory": "./dist",
       "not_found_handling": "404-page"
     },
     "routes": [
       {
         "pattern": "getfierro.com",
         "custom_domain": true
       }
     ]
   }
   ```

   Key decisions:
   - `not_found_handling: "404-page"` serves a 404 page for missing routes (Astro generates one)
   - `custom_domain: true` in routes tells Cloudflare to create/manage the DNS record automatically
   - No Worker code -- this is pure static asset serving

3. Add deploy script to `package.json`:
   ```json
   "deploy": "astro build && wrangler deploy"
   ```

4. Do NOT install `@astrojs/cloudflare` adapter -- it's only needed for SSR. This is a purely static site.

Note on FNDN-05 (images via Astro optimization): The `src/assets/` directory already contains `logo.svg` and `favicon.ico`. In Phase 1, no additional images are needed beyond these. The Astro `<Image>` component is available for use in Phase 2+ when product screenshots are added. The pattern is established: images go in `src/assets/`, imported via Astro components for automatic WebP/AVIF optimization.

   ```typescript
       await expect(mobileMenu).toBeVisible();

       await page.locator('#menu-toggle').click();

       await expect(menu.locator('a[href="/why-fierro"]')).toBeVisible();

       await expect(menu.locator('a[href="https://app.getfierro.com/signup"]')).toBeVisible();
     });

     test('mobile menu touch targets are at least 44px', async ({ page }) => {
       await page.goto('/');
       await page.locator('#menu-toggle').click();

       const links = page.locator('#mobile-menu a');
       const count = await links.count();
       for (let i = 0; i < count; i++) {
         const box = await links.nth(i).boundingBox();
         expect(box?.height).toBeGreaterThanOrEqual(44);
       }
     });

     test('mobile menu close button works', async ({ page }) => {
       await page.goto('/');
       await page.locator('#menu-toggle').click();
       const mobileMenu = page.locator('#mobile-menu');
       await expect(mobileMenu).toBeVisible();

       await page.locator('#menu-close').click();
       await expect(mobileMenu).toBeHidden();
     });
   });
   ```

   ```typescript
     test('body uses DM Sans font family', async ({ page }) => {
       await page.goto('/');
       const fontFamily = await page.locator('body').evaluate(
         el => getComputedStyle(el).fontFamily
       );
       expect(fontFamily).toContain('DM Sans');
     });

     test('body has off-white background color', async ({ page }) => {
       await page.goto('/');
       const bgColor = await page.locator('body').evaluate(
         el => getComputedStyle(el).backgroundColor
       );
       // Off-white should not be pure white (rgb(255,255,255)) or transparent
       expect(bgColor).not.toBe('rgba(0, 0, 0, 0)');
       expect(bgColor).not.toBe('rgb(255, 255, 255)');
     });

     test('body has gunmetal text color', async ({ page }) => {
       await page.goto('/');
       const textColor = await page.locator('body').evaluate(
         el => getComputedStyle(el).color
       );
       // Gunmetal is a very dark color, RGB values should be low
       expect(textColor).not.toBe('rgb(0, 0, 0)');
     });
   });
   ```
