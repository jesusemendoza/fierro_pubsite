**src/components/landing/FeatureRow.astro:**
A reusable component for a single feature row with alternating layout.

Props interface:
```typescript
interface Props {
  title: string;
  description: string;
  bullets: string[];
  reverse?: boolean;
}
```

Structure:
- Outer div with `data-animate` attribute
- Flex layout: `flex flex-col gap-8 lg:items-center lg:gap-16`
- Direction controlled by reverse prop: `${reverse ? 'lg:flex-row-reverse' : 'lg:flex-row'}`
- IMPORTANT: Default flex-col stacking is always text-first, visual-second (regardless of desktop direction). The reverse prop ONLY affects lg: breakpoint direction.
- Text column (flex-1):
  - h3: text-2xl font-bold text-gunmetal for title
  - p: mt-3 text-lg text-concrete-gray for description
  - ul with mt-4 space-y-2 for bullets
  - Each bullet: flex items-start gap-2, with a Molten Orange checkmark span and text-concrete-gray span
- Visual column (flex-1): `<slot />` for SVG illustration
